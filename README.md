# Monopoly des Services

Application locale React + TypeScript + Vite pour animer une partie de **Monopoly des Services**.

## Démarrage local

```bash
npm install
npm run dev
```

## Fonctionnalités incluses

- écran d’accueil et paramétrage des joueurs,
- plateau de jeu avec cases métier,
- suivi automatique des clients,
- gestion des pièces service et des enseignes complètes,
- sauvegarde locale via `localStorage`,
- écran de fin quand la banque est vide.
