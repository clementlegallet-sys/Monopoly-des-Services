# Visual specification

Use the reference images in this repository to improve the visual fidelity of the game.

Goals:
- make the board visually closer to the original Monopoly des Services
- keep the same overall board-game spirit
- preserve the main color families used in the original game
- make service tiles and card categories easy to distinguish
- keep the UI professional, playful, and suitable for team training
- do not turn the interface into a generic dashboard
